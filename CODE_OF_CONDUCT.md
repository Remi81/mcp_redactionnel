# Code of Conduct

Ce projet suit le [Contributor Covenant](https://www.contributor-covenant.org/). En participant à ce projet, tu acceptes de respecter le code de conduite suivant :

- Sois respectueux et constructif.
- Pas de harcèlement, d'insultes, ni de discrimination.
- Respecte la vie privée et les contributions des autres.

En cas de comportement inapproprié, ouvre **une issue privée** via l'interface de *Security* du dépôt ou utilise le mécanisme privé de signalement GitHub.

Les contrevenants pourront être exclus du projet à la discrétion des mainteneurs.
